超声电机

简介：主要记录阅读文献里有关超声电机的有关内容，以备查用。

1.《A micro ultrasonic motor using a micro-machined cylindrical bulk PZT transducer》微型超声电机
作者：Takefumi Kanda

内容：微型超声电机的设计，外壳不能被超声波激励，采用对照实验，采用微加工制造了超声波微型电机的换能器，在一定的电压和预载荷的条件下，可以输出一定的转速和转矩，但达不到驱动微机构的要求，提高预载机构和改善定子与转子之间的接触条件是关键。


2.《A Newly Developed Linear Motor-Driven Aerostatic X-Y Planar Motion Table System for
Nano-Machining》
作者：H.Shinno

内容摘要：为了适应纳米级制造加工的要求，设计一种新型直线电机驱动的空气静力学二维平面工作台。
1.介绍
为纳米加工开发了一款新型的无传感器监控功能的XY平面运动工作台。
2 平面运动工作台的开发
2.1 工作台的概念设计
系统使用空气静力学轴承和音圈电机，使工作台处于完美的非接触环境中，消除了各种非线性现象，存在的误
差因素最小化。通过加工力的反馈可以得到更好的控制性能。


2.2 配置及结构
电机的轭及线圈固定在基座，定子固定在工作台，这样移动工作台不存在粘滑、反作用力和热漂移等非线性
现象。整个系统由主动隔振系统支撑。


2.3 驱动平台理论设计
采用如图所示的对称系统，力总是作用于物体的中心，相互之间的影响及其微小。


2.4 工作台运动控制系统
采用全闭环控制系统，PID,加速度前馈补偿器。理论上系统静态刚度无限大。使用干扰观测器的加工力无传感
器监控功能安装在每一个轴的控制系统中。该系统利用驱动力指令和测量的工作台位置精确计算加工力，即使在加
工过程中也能实现纳米运动控制。


3 定位性能
测试无负载条件下的的定位性能，驱动特性是相对独立的。跟踪误差甚至在变化的加速度下小于1nm，在工作
台的运动起点没有反向间隙等非线性现象。可以实现无象限误差和反演误差的平稳运动，并且此工作系统保证了XY运动的平稳性。因此，根据结果性能表现良好，不需要再进行复杂的调优工作。


4 加工性能


加工实验结果证明研制的工作台有较好的加工性能。
5 结论
1):研制了一个加工力的无传感器反馈功能的工作台系统。
2):在纳米级的加工条件下，工作台系统具有在XY平面内进行纳米定位的能力。
3):对于脆硬性材料具有高刚度和高稳定度。
